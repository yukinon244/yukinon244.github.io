我永远喜欢豆腐脑（甜的） 2016秋

划水划过了10个实验...

虽然说课程表上写的是周二周四下午做实验，然而实验基本都是放到周末做的...10月上旬有两节绪论课，安排在周二和周四的下午。

有两个需要计算的实验还有8个演示型实验...

# 分子电子结构的量子化学计算：要写预习然后现场交计算结果，用Gaussian算水分子的能量。然而每一次我用自己的电脑跑Gaussian总会出现内存错误这种神奇的问题...

# 碳纳米管的第一性原理计算：简易的VASP教学，最后要写实验报告。要画DOS之类的东西。不过服务器上有自动画DOS的脚本

(推荐一个画图软件VESTA：http://jp-minerals.org/vesta/en/ 画出的图好看啊！！）

# 噻吨酮与二苯胺的电子转移反应的激光闪光光解研究：印象中是小组交报告的实验

# 拉曼光谱的测量和振动光谱的理论计算：一个小组交一份报告。中途会演示GaussianView怎么用=。=

# 动态光散射测量粒子的大小及其分布：一个要写实验报告的演示型实验。

# 碳纳米管负载CuO-CeO2催化剂的制备和表征/碳纳米管负载CuO-CeO2催化剂的富H2条件下CO优先氧化反应性能：一个小组交一份报告的演示型实验

# 可调谐式半导体激光吸收光谱仪及其应：一个需要写报告的演示型实验

# 循环伏安法以及影响电化学测量结果的因素分析&旋转圆盘电极研究温度对氧还原活性的影：一个下午就可以做完的实验。。一个小组交一份报告

# 表面分子结构的和频光谱表征：也是一个需要写报告的实验。

总之报告可以放到年底一起交...大概到10月底才会开始安排实验，12月中旬就可以做完所有的实验。
