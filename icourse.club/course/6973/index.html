这是一门神奇的课。

讲课顺序大概是这样的：高等无机的讲课内容 -> 计算材料学概述 -> 对称性（分子点群/晶体的对称性) -> 固体科普知识（能带相关、磁性） -> 催化科普知识 -> 无机功能材料 -> 配位化合物 -> 原子簇化合物/超分子....

课件内容万年不变，并且对打印店特别不友好因为背景是深蓝色的还是pdf的格式的没法改……我觉得认真听课的话收获好像还是挺大的，然而由于某些原因…听课存在一定困难。

中途有一次作业，今年的是写对称性在你的研究领域中的应用。还有一次点名，至于什么时候点名，看他的心情。

考试开卷，基本都可以从他上课的ppt中找到答案，给分充满了爱与希望。。。

（另外赠送一份打印了的高等无机的课件。。有需要的可以联系我=。=）
