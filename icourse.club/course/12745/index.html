曾老师是个好老师

02/04/2022 02:43:38 0 0

立即登录，说说你的看法

呜哩哇啦 2021秋

放假了，来给曾文老师好评。

上学期选课的时候，看到评课社区有的评价说曾老师，给分不是很好，评分标准有些迷惑，我选课是内心是慌张的，不过一学分的体育课也无所谓了，就选了曾老师班（不过好像是捡漏，有一个人从曾老师班上退出来了）

开学第一次课，老师会问大家有没有基础，会不会游泳啊。理论上会游的同学也不不应该来基础班吧。当然，你会游泳却装作不会，也是不行的，老师能看出的。

游泳基础班教的蛙泳，先学会在水里漂，然后蹬腿，之后手臂，最后是换气，最后三次课教反划（准确来说是自己学，这个动作好像没法教，多喝几口水就会了），下深水区。一学期下来，班上的同学都能学会，时间是很充裕的，曾老师会亲自指导学的慢的同学，帮忙纠正错误姿势什么的。

考试：秋季学期体育课都要考12分钟跑，和25m游泳（速度和技术评价（五等级制），姿势是否标准），没有考反划。刚开始就会的同学考50m。

大家的考试成绩会当堂公布，很透明，老师应该看姿势的标准程度和进步水准比较多，给分很好。

反正上课要注意联系老师反复强调的点吧，腿部动作，手臂动作，尽管游泳时会游淹不死就行，但是给分上老师还是很看重这个的。

老师第一次课就说我们不点名，建有微信群，可以在里面请假，老师也都会批准。但每周五下午开始，就开始有人请假，之后又不少人跟着请假，一节课5，6个人不来，属实难绷。
