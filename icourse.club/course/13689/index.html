第一次写点评（

大一上选这门公选课主要出于好奇吧，至少我那时想编程语言什么的我还是会一点的，这门课应该是比较简单的。事实也确实是这样。

课程大概是普及 VR 相关知识 + Unity & C# 相关。照着视频做，结果不会太糟糕，就算不会 C# 也没关系，只要之前有一门语言基础 + 会使用搜索引擎，语言使用不会成为问题。

课程最后评分（二分制）极水，其实只要做完线上部分，线下再上一次课就不会挂。我还去把线上视频里面的滚小球 Demo 修改出新游戏呢……似乎只有我和我舍友做新项目的……之后课程评分机制据说会改，虽然跟我已经没啥关系了。

当然这门课槽点也不少：

  1. SDK 更新速度很快，这就造成视频中使用的 SDK 版本与最新版本差别很大，并且在最新版本的 Unity 上使用旧版 SDK 也有一点问题。所以为了上这门课，你不得不去找旧版本 SDK 和旧版本的 Unity……
  2. 我能说很多人连 VR 设备都没有吗？为了一个你根本不知道长什么样子的平台上面做游戏，当然不爽。虽然我其实也没有设备，最后是拿舍友的 VR 眼镜调试的。
  3. 高校邦（就是这门课的慕课平台）网页版操作很难做到一边看视频一边照着做，因为当浏览器窗口失去焦点的时候，视频是自动暂停的。当然，你可以使用（经常崩溃的）手机 App 看，或者审查元素（嘿嘿）。

大概感觉就是：结束之后想用 Unity 写点正经游戏，C# 这门语言好像也不错啊，但我对 VR 依旧毫无兴趣。

希望以后能专门有一门 Unity 的公选课，毕竟对于大部分没有设备的人来说，做出一个可以直接在电脑上运行的游戏的成就感比这要大得多了。
